Sticky Table of Contents — Responsive Layout (HTML + CSS)

This project demonstrates a clean, semantic, and accessible layout consisting of:

    A main article
    A right sidebar Table of Contents (TOC)
    A sticky TOC that remains visible while scrolling
    A fully responsive, mobile-friendly layout using CSS Grid only (no JavaScript)

Features
    Semantic Structure like <main>, <article>, <nav>, <section>, <figure>, <img>, headings (h1–h3).

    In-page anchor links using unique id attributes.

    TOC links scroll smoothly to the correct sections.

    Responsive Layout Using Grid.

    Desktop: Two-column layout.

    Mobile: Automatically collapses to a single column.

    TOC moves above the content on small screens.

    Meaningful alt text for images.
